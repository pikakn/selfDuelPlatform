遊戯王向け一人回し用のインターフェイス「SelfDuelPlatform」です。
現物のカードでの一人回しと同じように、カードをルールを無視して動かせ、墓地や除外ゾーンなどの確認もできるように設計しました。
手札のゾーンや、使いそうなカードの仮置き場も用意してあります。
右上のボタン「code」から「dounload zip」を選択し、zipファイル(selfDuelPlatform-mainという名前がついています)をDLしてください。
お好きな場所に展開してください。
フォルダ内のselfPuelPlatform.htmlを開くことで、本アプリがブラウザで起動します（コマンド操作はありません）。
詳しい使い方はアプリ内に記載してあります。
画面が見づらい場合、ブラウザのズーム機能をお試しください。

本フォルダの中身は、imageフォルダの中を除き変更しないでください。
画像の読み込みなどに影響が出ます。
